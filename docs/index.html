6大慢病全周期管理

基于循证医学指南，覆盖高血压、糖尿病、冠心病、COPD、慢性肾病、脑卒中，实现从筛查到长期随访的完整闭环。

🫀 高血压管理

基于Framingham评分 + 中国人群修正因子，10年心血管事件风险评估。阶梯式用药推荐，家庭血压自动采集。

血压达标率 靶器官监测

🍬 糖尿病管理

UKPDS风险引擎预测微血管/大血管并发症。HbA1c趋势分析，AI营养师个性化饮食方案。

血糖监控 并发症筛查

❤️ 冠心病管理

GRACE评分 + SYNTAX评分综合评估。ABCDE二级预防方案自动提醒，心脏康复分期运动处方。

LDL-C达标 紧急预警

🫁 慢阻肺管理

GOLD分级 + mMRC + CAT评分综合评估。吸入装置使用指导，急性加重早期识别。

肺功能监测 戒烟支持

🫘 慢性肾病管理

KDIGO风险矩阵分层。eGFR趋势分析，肾功能不全时药物剂量自动调整提醒。

透析准备 饮食管理

🧠 脑卒中管理

CHA2DS2-VASc + ABCD2评分。复发预防、功能康复、卒中后抑郁筛查与干预。

二级预防 功能康复
