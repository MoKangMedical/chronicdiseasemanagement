# 栖霞区公开资料整理

这个目录收录当前项目已经引用或准备引用的栖霞区公开资料，目标是把“页面展示口径”和“系统可接入字段”放在同一处维护。

## 文件说明

- `qixia-public-data.json`
  - 结构化数据总表
  - 包含人口底数、年龄结构、性别结构、健康指标、可接入系统资产与来源清单
- `sources-index.md`
  - 面向产品和实施同学的可读索引
  - 按来源说明“这份资料能进系统哪里”

## 当前整理范围

1. 栖霞区 2024 国民经济和社会发展统计公报
2. 栖霞区 2025 年民生服务报道
3. 栖霞区 2026 年卫生健康工作会议报道
4. 栖霞区 2024 年卫生健康工作会议报道
5. 栖霞区紧密型城市医疗集团成立报道
6. 栖霞区第七次全国人口普查年龄金字塔

## 使用建议

- 首页“公开区情口径”面板，直接读取 `qixia-public-data.json`
- 区级覆盖率、医院覆盖率、基层触达率等指标，统一使用这里的公开底盘口径
- 病种比例、治疗包比例、起效率这类业务指标，仍以平台纳管数据为准，不要混用公开资料
